Hydra
=====


Hydra is an open-source Python framework that simplifies the development ofresearch and other complex applications. The key feature is the ability todynamically create a hierarchical configuration by composition and override itthrough config files and the command line. The name Hydra comes from itsability to run multiple similar jobs - much like a Hydra with multiple heads.

https://hydra.cc/
