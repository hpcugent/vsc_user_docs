metaWRAP
========


MetaWRAP aims to be an easy-to-use metagenomic wrapper suitethat accomplishes the core tasks of metagenomic analysis from start to finish:read quality control, assembly, visualization, taxonomic profiling, extractingdraft genomes (binning), and functional annotation.
