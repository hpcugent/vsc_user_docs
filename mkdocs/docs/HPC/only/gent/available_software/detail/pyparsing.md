pyparsing
=========


The pyparsing module is an alternative approach to creating andexecuting simple grammars, vs. the traditional lex/yacc approach, or the use ofregular expressions. The pyparsing module provides a library of classes thatclient code uses to construct the grammar directly in Python code.
