CFDEMcoupling
=============


CFDEMcoupling is an open source CFD-DEM engine. It provides the possibility to couple the DEM engine LIGGGHTS to a CFD framework.

https://www.cfdem.com/cfdemrcoupling-open-source-cfd-dem-framework
